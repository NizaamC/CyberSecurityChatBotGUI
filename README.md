# CyberSecurity Awareness Chatbot

## Description
A GUI-based cybersecurity awareness chatbot developed in C# using WPF and .NET 8.

## Features
- GUI chat interface
- Voice greeting
- Keyword recognition
- Random chatbot responses
- Sentiment detection
- Memory and recall
- Conversation flow
- Delegates and collections
- Error handling

## Technologies Used
- C#
- WPF
- .NET 8
- GitHub
- GitHub Actions

## Cybersecurity Topics
- Password safety
- Phishing
- Privacy
- Online scams

## How to Run
1. Open project in Visual Studio
2. Build and run the application
3. Interact with the chatbot
